# Cloud-Music Player

### Running the code 
Just go into the code directory and type 
```
python manage.py runserver
```
"Cloud-Music player" app will start on 127.0.0.1:8000 (Local Address)

### Features

- Add new albums
- Add songs to Albums
- Search for songs and Albums
